#### 1. API 返回 Mock 数据
- **状态**: 完全阻塞 API 功能
- **影响**: API 调用返回假数据
- **修复**: 连接到真实 TmCore
- **工作量**: 2-3 天
- **文件**: `apps/api/src/services/task.service.ts`

#### 2. 缓存不一致
- **状态**: 数据一致性问题
- **影响**: CLI 和 API 显示不同数据
- **修复**: 文件监听 + 自动缓存失效
- **工作量**: 1-2 天
- **文件**: `apps/api/src/middleware/cache.middleware.ts`

#### 3. 文件并发写入风险
- **状态**: 数据丢失风险
- **影响**: 多进程写入导致损坏
- **修复**: 文件锁 + 原子写入
- **工作量**: 1-2 天
- **文件**: `packages/tm-core/src/modules/storage/`

### P1 优先级 - 架构质量 (第2-3周)

#### 4. 存储抽象不完整
- **状态**: 技术债 - 未来阻塞器
- **影响**: 无法迁移到数据库
- **修复**: 创建 ITasksStorage 接口
- **工作量**: 3-5 天
- **文件**: `packages/tm-core/src/modules/storage/`

#### 5. 无跨层变更通知
- **状态**: 功能阻塞器

- **修复**: EventBus + 事件监听
